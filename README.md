# agrinho2026# Inteligência e Biotecnologia no Futuro do Gado de Corte

## Objetivo

Apresentar de forma moderna, acessível e responsiva como a Inteligência Artificial e a Biotecnologia estão transformando o agronegócio brasileiro, destacando sustentabilidade, inovação e produtividade no campo.

---

## Tecnologias Utilizadas

- HTML5 Semântico
- CSS3
- JavaScript Vanilla
- CSS Grid
- Flexbox
- SpeechSynthesis API
- Design Responsivo

---

- Banner futurista

---

## Como Utilizar

1. Salve os arquivos:
   - index.html
   - style.css
   - script.js

2. Coloque suas imagens na mesma pasta:
   - Foto1.png
   - Foto2.png
   - Foto3.png

3. Abra o arquivo index.html no navegador.
